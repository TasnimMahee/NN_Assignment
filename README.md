# Neural Network Assignment Repository

This repository contains solutions for two machine learning problem sets, covering both deep learning and traditional machine learning approaches.

---

## Problem Set 1  
**Pneumonia Detection using Convolutional Neural Networks (CNN)**  

This task involves building a CNN model to classify pediatric chest X-ray images into two categories: *Pneumonia* and *Normal*.

---

## Problem Set 2  
**Bank Term Deposit Prediction using Logistic Regression**  

This task focuses on developing a classification model to predict whether a customer will subscribe to a term deposit based on demographic and banking data.
